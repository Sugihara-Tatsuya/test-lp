## 矛盾したときの裁定ルール（実績のある判断）

1. **口頭発言より表形式の生データ**。議事録の「MRR今1億くらい」より、値上げシミュレーション表の 99,799,899円 を採用した。口頭値は範囲が曖昧（全社か受発注のみか不明）なことが多い。
2. **原本シートより「社内で実際に管理されている粒度」**。原本のQ目標内訳（アップセル60/クロスセル70）は実在しない目標で、実際のKGIは合算エクスパンション130万円/月だった。
3. **報告慣行そのものを疑う**。Slack報告の帰属月ズレは、12月実績9,596万円が経営会議の9,587万円・プロダクト別シートの2025年末値と整合することの突合で確定させた。1点の食い違いは、複数の既知値との整合で裁定する。
4. **議事録の混乱はそのまま記録して裁定はメインで**。言い直し（「1740万→150万」）や隠語（**「肉」=4Q**）が生のまま残っている。サブエージェントには憶測補完を禁じ、原文のまま報告させる。
